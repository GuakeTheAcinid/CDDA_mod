This mod is made to make Cataclysm: Dark Days Ahead closer to what I would like to see in game. Also I was wondering where did interesting features gone, from what I heard people had in-game and what I've seen myself, so I brought some of them back.

This mod adds miscellaneous staff which I personally like.

All Original stuff is located in "guakes_adjustments" folder, and all stuff returned from previous versions are in "bring_be_back" folder.
